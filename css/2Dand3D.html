  • 2D转换
    • transform:translate;
    • 总结：
      • 通过translate可以移动元素位置
      • translate让元素位置移动的时候，是相对元素自己本身原来 的位置（位置的数字可以设置百分比
      • translate最多能设置两个值，第一个代表水平方向，第二值代表垂直方向的位移
      • translate的方向：正数代表正方向，负数代表反方向
    • 旋转：ransform:rotate（--deg）
      ransform-origin可以改变旋转的原点
      • 可以设置关键字left|
      • 也可以设置具体数值
      • 2D旋转transform:rotate（）
      • 总结：
        • 旋转的单位是deg
        • 旋转如果设置的是正数，那么顺时针旋转，反之逆时针旋转
        • 只能设置一个参数
        • 默认元素是绕着自己的中心点位置旋转
    • 缩放(scale)
      • 总结
        • 缩放提供的是两个参数，代表的是倍数关系
        • 如果设施一个值，那么代表当前宽度和高度同事放大或缩小相应的倍数
        • 如果设置两个值，那么第一个值代表水平方向的倍数， 第二值代表垂直方向的倍数
        • 如果设置的数字是0-1之间的值，代表的是缩小的倍数。
        • 不能设置负数
    • 倾斜（skew）
      • 总结：
        • 设置第一个值代表水平方向，设置第两个值代表垂直方向。
        • 设置正数是往左移，这是负数往右移。

  • 动画：animation
    keyframes
  • 动画是复合属性
    用法：
    • 先通过animion来制定一个动画集@keyframes 自定义个名称{ form(开始){}to(结束){} }
    • 标签通过animation调用动画集
      • animation-name设置动画序列名称
      • animation-duration动画持续时间
      • animation-delay动画延时时间
      • animation-timing-function动画执行速度，linear、ease等
      • animation-play-state动画播放状态，running、paused等
      • animation-direction动画逆播，alternate等
      • animation-fill-mode动画执行完毕后状态，forwards、backwards等
      • animation-iteration-count动画执行次数，inifinate等
      • steps(60) 表示动画分成60步完成
      • 让动画暂停：animation-play-state:paused;
      • animation-fill-mode该属性必须与animation-iteration-count配合使用。animation-iteration-count属性不能为inifinate
      • 注意：一个元素可以同时调用多个动画集使用逗号隔开 调用动画集的时候，必须设置时间和动作
      • 百分比是相对动画执行的时间
  • 伸缩布局（弹性布局）：
    • 伸缩布局的步骤：
      • 先直接给父元素设置成伸缩盒子display: flex;
      • 伸缩盒子特点：
        • 伸缩盒子具有主轴（默认盒子方向从左向右）
        • 伸缩盒子还有侧轴（侧轴始终要垂直于主轴）
        • 在伸缩盒子中所有的子元素都要按照主轴的方向显示
      • 内容：

        • 主轴的方向：

          flex-direction调整主轴方向（默认为水平方向）row从左到右显示|row-reverrse从右到左显示|column从上往下|column-reverrse从下网上显示

        • 设置子元素在主轴的对齐方式：

          justify-content调整主轴对齐flex-start开始处|flex-end结束处|flex-center中间处|space-around环绕对齐|space-between左右两侧对齐中间的处正中间

        • 设置子元素在侧轴的对齐方式：

          align-items调整侧轴对齐align-items:flex-start|flex-end|center|stretch拉伸（默认值）

        • flex-wrap控制是否换行:

          nowrop不换行|wrap换行|flrx-start换行以后的对齐方式： align-content: |flex-end换行后的对齐方式底部|center中间|space-between|space-around|stretch(默认)
      • 伸缩盒子子元素的属性：order|flex|
        • order： 值越大在浏览器的显示位置越靠后
        • 如果父元素中有固定的大小的盒子，子元素如果设置了flex属性，那么flex该属性的值代表的意思是：占父元素减去
        • flex：控制盒子的伸缩比
